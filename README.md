# Machine-Learning

This project consist in the creation of a machine learning tool implememted in a web-based app using Django development framework. Here, users can access for information related to diabetes diagnostic and management from different perspective including socio-economic and purely medical.

To run the program one as to use the command 'pip install -r requirements' to install all the relevant libraries.

The Home page shows the diabetes death rate in a few places around the world, feel free to enter the entity in this format 'France' or 'Asia'.

In the predict section, one can use the diabetes prediction tool to see how socio-economic and medical factors shape the likelihood to suffer from diabetes (CDC Diabetes form UCI ML). The other predict section uses the Pima dataset to make diabetes prediction and is only relevant for women, in contrast to the other section that can be used for anyone.

The BMI calculator calculates the BMI of a person and provides a class category for that individual.

The 'Further Studies' section serves to collect data from anyone willing to provide some relevant information for future studies on diabetes management.

Data collected will be store form a 2 years and in regards to data protection individuals are not ask for their personal details ensuring anonymity with the data collected.
